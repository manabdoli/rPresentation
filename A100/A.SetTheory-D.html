A. Set Theory: Set Operations

Mansour Abdoli, PhD

Concept Overview

  • Union (\(A \cup B\))
  • Intersection (\(A\cap B\))
  • Difference (\(A-B\))
  • Universal set (U or S)
  • Complement (\(A^c = U-A\))

Union \(\cup\)

  • Any elements of two (or more) sets

  • \(A\cup B = \{ x | x\in A \text{ OR } x\in B\}\)

  • Example:

    • \(A = \{1, 2, 3\}, B = \{3, 4, 5\}\)

    • \(A \cup B =\) {1, 2, 3, 4, 5}

Intersection \(\cap\)

  • Common elements of two (or more) sets

  • \(A\cap B = \{ x | x\in A \text{ AND } x\in B\}\)

  • Example:

    • \(A = \{1, 2, 3\}, B = \{3, 4, 5\}\)

    • \(A \cap B =\) {3}

Q1.

What happens to the cardinality of union and intersection?

  1. There will be no difference.
  2. Cardinality of union and intersection goes up.
  3. Cardinality may increase for union but may decrease for intesection.
  4. Cardinality will increase for union and decrease for intesection.

Differnce \(A\setminus B\) or \(A-B\)

  • Elements in \(A\) that are not in \(B\)

  • \(A-B = A\setminus B= \{ x | x\in A \text{ BUT (AND) } x\notin B\}\)

  • Example:

    • \(A = \{1, 2, 3\}, B = \{3, 4, 5\}\)
    • \(A - B =\) {1, 2}

Universal Set and Complement

  • Universal Set \(U\): All relevant elements
  • Complement of Set \(A\): Difference of Universal and set \(A\)
    • \(A^c=A'=U-A\)
  • Example:
    • \(U=\{1, 2, ..., 6\}, A = \{1, 2, 3\}\)
    • \(A^c\) = Complement of A = {4, 5, 6}
